Prerequisites:
    - 16GB RAM or more
    - Internet connection
    - Python 3.6+
        - beautifulsoup4
	- nltk
	    - punkt
	    - stopwords
	- pandas
	- pyparsing
	- PySimpleGUI
	- regex
	- requests
	- sklearn
	- tkinter (on ubuntu I had to run: sudo apt install python3-tk)

Execute the python file in the extracted folder:
    For example, to execute with the system interpreter:
        1. python elgoog.py
        
    Otherwise, to execute in a virtual environment (e.g. pipenv):
        1. pipenv --python 3 run python elgoog.py
